XMLRPC Math

 
Run XMLRPC Server
 

Use below format to utilize the XMLRPC Math package:

var = math.function(arguments)

Example:

letter_H = math.strIndex("Hello",0)

 

Following functions are implemented:

add(a,b) - Adds two numeric values

sub(a,b) - Subtracts two numeric values

strIndex(str,n) - Returns the n'th character of string str

strCon(str1,str2) - Concentrates str1 and str2
